# UniFood App Icon Design

## Design Specifications

### Colors
- **Primary**: Green (#4CAF50)
- **Secondary**: Orange (#FF9800)
- **Background**: Gradient from Green to Orange
- **Text**: White (#FFFFFF)
- **Accent**: White with transparency

### Icon Elements
1. **Background**: Rounded rectangle with gradient (Green to Orange)
2. **Decorative Elements**: Three semi-transparent white circles
3. **Main Icon**: Restaurant/food service icon with:
   - White plate/container
   - Colorful food items (green, orange, red)
   - Fork and knife on sides
4. **App Name**: "UniFood" in bold white text

### Technical Specifications
- **Size**: 512x512 pixels (for app stores)
- **Format**: PNG with transparency
- **Style**: Modern, clean, professional
- **Theme**: University canteen/food service

### Usage
This icon should be used for:
- App launcher icon
- App store listings
- Splash screen
- Profile section branding
- Help section branding

### Design Principles
- **Simple**: Easy to recognize at small sizes
- **Memorable**: Unique color combination
- **Professional**: Suitable for university environment
- **Scalable**: Works at various sizes
- **Branded**: Consistent with UniFood identity

### Color Palette
```
Primary Green: #4CAF50
Secondary Orange: #FF9800
White: #FFFFFF
Light Green: #81C784
Light Orange: #FFB74D
```

### Implementation Notes
- Use the Flutter AppLogo widget for consistent branding
- Ensure high contrast for accessibility
- Test at various sizes (16px to 512px)
- Maintain aspect ratio in all implementations
